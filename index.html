About Me


I grew up and graduated high school from a small town in Wisconsin called Rice Lake. In high school I was always active, participating in the Football, Swimming, and Track teams. Once the rotational program was completed I stayed with the position of Operations Support Business Analyst. In this role I communicated with business users when they had trouble with our in-house application for buyers to purchase new material and view open orders. During my time in this role I also acted as project manager for the upgrade of a planning software used by customer service representatives.

Projects

Python 2D Scroller

This is a 2D side scroller game that has kept me very busy! I am still in the stages of creating the physics and general engine of the game. I am using the following libraries:


  • pyGame - A very popular and widely used library that helps create computer graphics and sounds in the Python language
  • pyTMX - A library that helps sort through tilesets to create maps for a game
  • pyScroll - This library lets you scroll through maps that do not fit on a single screen

I am not nearly completed with this game and am looking forward to continuing its development. At its current stage, the game drops you into a small map, where you can control your character with the arrow keys. The character is generated from a spritesheet that makes the walking animation. Not only can you walk, but your character also has a gun that he can shoot. Beware of the one enemy on the map though, make sure to take him out before he hits you!


Enhancements in the near future include:


  • Health system
  • Ammo system
  • Intelligent AI that fights back
  • Different weapons
  • Level creation

Android Game

This project has been an absolute blast to work on. I have always been curious how mobile games are made, so I figured to best way to find out was to make one! This game is still in its infancy stages but I am happy with what I have accomplished and learned so far.


Once the game loads you find a makeshift controllor at the bottom of your phone. The left rectangle is for moving your ball, and the right is for aiming. As you move your finger in the aiming control box, you will notice that a crosshair moves in a perfect circle around your ball. You can also fire out a projectile once you have set your sights. Obviously the game has no objective yet, but I have plans to create a full game from this concept!


Enhancements in the near future include:


  • Smoother vertical scrolling
  • Border detection
  • Enhanced graphics (Everything is placeholder at the moment)
  • Creating the game objective (It's a secret for now!)

This is game was created with the libGDX library - a popular library for creating games in Java

News Website Using PHP

At the moment, this project is admittedly a mess! The idea of this website is for users to read up to date articles and for authors to easily be able to log on and create content. This experience has been great as it has taught me a lot about PHP and everything that is possible with it. The main features currently include:


  • Dynamic page creation - The pages are not static, they are generated by pulling the most up to date articles
  • A login system - The login system checks the username and password with an SQL database
  • A homemade CMS (Content Management System) - This allows authors to login with a username and password, and write an article which is then stored in a database

Unfortunately, the site is currently under construction, so showing all the featues is a bit difficult. I plan to keep you up to date with its progress, and you can view the code on my gitHub page!
